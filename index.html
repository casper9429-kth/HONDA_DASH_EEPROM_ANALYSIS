Quick Reference

Key differences between diesel and petrol EEPROM variants.

Diesel (SMJ) Petrol (SNA / SNB)
Header BA 65 FF 77
Odometer formula km = value × 31 km = value × 32
Location 0x180–0x1BF 0x180–0x1BF
Global checksum None None
